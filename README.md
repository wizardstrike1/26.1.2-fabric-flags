# Flag Quiz Fabric Mod (26.1.2)

Client-side Fabric mod for Lunar/Fabric 26.1.2.

## What it does

- Run `/flagquiz` to toggle the quiz on/off.
- While enabled, every 30 seconds a country flag popup appears.
- Type the country name and submit.
- If your answer is wrong, chat shows the correct country.

## Build

```powershell
.\gradlew.bat build
```

Built jar:

`build\libs\flag-quiz-fabric-1.0.0.jar`

## Install in Lunar profile

Copy the built jar into:

`C:\Users\Leo\.lunarclient\profiles\vanilla-qol\mods`
